# Next.js 现代化电商平台

这是一个从零搭建的高性能电商应用，专注于极致的用户体验与响应速度。

<img src="https://picsum.photos/seed/ecommerce/800/400" alt="电商首页" width="100%" />

核心特性包含：
- 服务端渲染 (SSR) 与流式加载
- 无缝集成的 Stripe 支付系统
- 完美适配各端屏幕的自适应交互
