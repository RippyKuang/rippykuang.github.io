# 如何发表与编写项目介绍

# 如何发表

将项目介绍文章放入projects目录下，即可完成发表

# 如何编写

项目介绍同样包含两个部分，**相关信息**与**正文**

## 相关信息

``` yaml
---
title: "Awesome Project"
startDate: "2023-01-01"
endDate: "2025-01-04"
hasGithub: true
hasLiveDemo: true
excerpt: "This is a mock project used to test pagination on the Projects page."
repo: "https://github.com/exp/project"
link: "https://project.example.com"
---
```

## 正文

正文的编写与*post*相同
